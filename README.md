## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/realcapobambino/portfolio.git

2. Navigate to the project directory:

    ```bash
    cd portfolio

3. Install all the dependencies:

    ```bash
    bun install

4. Start the development server

    ```bash
    bun run dev
